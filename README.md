## ğŸ”¬ Analytical Approach
The project follows a **hypothesis-driven and evaluation-first methodology**:

1. **Exploratory Data Analysis (EDA)**  
   - Data distribution analysis  
   - Trend & anomaly detection  
   - Early risk identification  

2. **Data Cleaning & Preprocessing**  
   - Missing value treatment  
   - Outlier handling with domain logic  
   - Consistency and validation checks  

3. **Feature Engineering**  
   - Process, environmental, and supplier-level features  
   - Interaction and stability indicators  

4. **Predictive Modeling**  
   - Regression-based shelf-life prediction  
   - Tree-based and ensemble models for non-linear behavior  
   - Hypothesis testing through model interpretability  

5. **Model Evaluation & Risk Assessment**  
   - Cross-validation and temporal validation  
   - Error impact and risk-aware evaluation  
   - Business-aligned performance interpretation
